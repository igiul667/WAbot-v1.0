## Language file format
This files must contain information according to the following scheme based in english:
| line |       content       |
|------|:-------------------:|
|    0 | Error message       |
|    1 | on youtube          |
|    2 | Looking for audio   |
|    3 | Looking for video   |
|    4 | Here's what I found |
|    5 | TTS says:           |
|    6 | Looking for image   |
|    7 | on google           |
